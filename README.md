# 微信小程序打卡工具 - 使用说明

## 项目结构

```
checkin-app/
├── app.js                    # 小程序入口文件
├── app.json                  # 小程序全局配置
├── app.wxss                  # 全局样式
├── project.config.json       # 项目配置文件
└── pages/
    └── index/
        ├── index.js         # 首页逻辑
        ├── index.json       # 首页配置
        ├── index.wxml       # 首页结构
        └── index.wxss       # 首页样式
```

## 功能特性

✅ **一键打卡** - 支持上班、下班、外出、返回四种打卡类型
✅ **自动定位** - 获取当前位置经纬度
✅ **地址解析** - 通过腾讯位置服务解析详细地址（需配置Key）
✅ **自定义备注** - 每次打卡可添加备注信息
✅ **历史记录** - 显示最近50条打卡记录
✅ **数据统计** - 显示今日和本周打卡次数
✅ **本地存储** - 数据保存在手机本地
✅ **导出功能** - 支持导出CSV格式到剪贴板
✅ **记录管理** - 支持删除单条记录或清空所有记录

## 使用步骤

### 1. 导入项目

1. 打开微信开发者工具
2. 选择"导入项目"
3. 选择本项目文件夹
4. AppID 可使用测试号或填写自己的小程序AppID

### 2. 配置腾讯位置服务（可选）

如果需要地址解析功能，需要申请腾讯位置服务Key：

1. 访问 [腾讯位置服务](https://lbs.qq.com/)
2. 注册并登录控制台
3. 创建应用并获取Key
4. 在 `app.js` 中填写Key：
   ```javascript
   globalData: {
     qqMapKey: '你的腾讯位置服务Key'
   }
   ```
5. 在 [微信公众平台](https://mp.weixin.qq.com/) 的小程序管理后台，添加 `apis.map.qq.com` 到 request 合法域名

如果不配置Key，将显示经纬度坐标作为地址。

### 3. 编译运行

1. 点击"编译"按钮
2. 在模拟器或真机上预览
3. 首次使用需授权位置权限

## 功能说明

### 打卡类型

- **上班** - 上班打卡
- **下班** - 下班打卡
- **外出** - 外出打卡
- **返回** - 返回打卡

点击相应按钮选择打卡类型，选中的类型会高亮显示。

### 定位功能

- 打开小程序自动获取位置
- 点击"刷新定位"可重新获取
- 首次使用需授权位置权限
- 定位成功后可看到详细地址或经纬度

### 打卡操作

1. 选择打卡类型
2. 等待定位成功
3. 可选：输入备注信息
4. 点击绿色"打卡"按钮
5. 打卡成功会有振动反馈

### 记录管理

- **查看记录** - 打卡后自动显示在列表中
- **删除记录** - 点击记录右侧的"✕"按钮
- **清空记录** - 点击右上角"清空"按钮
- **导出记录** - 点击右上角"导出"按钮，数据将复制到剪贴板

### 数据存储

- 所有数据存储在手机本地（wx.setStorageSync）
- 最多保存最近50条记录
- 清除小程序缓存会删除所有数据

## 注意事项

1. **位置权限** - 需要在小程序设置中开启位置权限
2. **腾讯位置服务** - 免费版有调用次数限制（每日5000次）
3. **数据备份** - 建议使用导出功能定期备份数据
4. **测试号限制** - 使用测试AppID时部分功能可能受限

## 自定义开发

### 修改配色

编辑 `pages/index/index.wxss`，修改以下颜色值：

- 主题色：`#07c160` (绿色)
- 危险色：`#ff4d4f` (红色)
- 背景色：`#f6f6f6` (浅灰)

### 添加云端同步

如需云端同步功能，可以：

1. 使用微信云开发
2. 接入自己的后端API
3. 使用第三方BaaS服务

修改 `pages/index/index.js` 中的 `doCheckin` 和 `loadRecords` 方法。

### 增加打卡类型

编辑 `app.js` 中的 `checkinTypes` 数组，并在 `index.wxml` 中添加对应的视图。

## 常见问题

**Q: 定位失败怎么办？**
A: 检查是否开启了位置权限，可在小程序设置中重新授权。

**Q: 如何查看历史记录？**
A: 打卡后记录会自动显示在页面底部，最多显示最近50条。

**Q: 数据会丢失吗？**
A: 数据存储在本地，清除小程序缓存会丢失数据，建议定期导出备份。

**Q: 可以修改打卡时间吗？**
A: 当前版本不支持修改，使用实时时间自动记录。
